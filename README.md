# 🎯 Guess The Number Game

A fun Python game where you guess a random number between 1 and 100.

## 🚀 Features
- Random number every time
- Hints (Too high / Too low)
- Tracks number of attempts

## 💻 How to Run
1. Install Python
2. Run the file:
   python guess_game.py
